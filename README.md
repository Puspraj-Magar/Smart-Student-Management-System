# 🎓 Smart Student Management System

A console-based application to manage student records and academic performance using Java, MySQL, and JDBC.

---

## 🚀 Features
- ➕ Add, update, delete students
- 📋 View student details
- 📝 Add and manage grades
- 📊 GPA calculation
- ✅ Input validation & error handling

---

## 🛠️ Tech Stack
- Java
- JDBC
- MySQL

---

## 🗄️ Database
- Relational schema with foreign keys
- Constraints for data integrity
- SQL file included in `/database/schema.sql`

---

## ▶️ How to Run
1. Clone the repository
2. Import into IntelliJ / Eclipse
3. Setup MySQL database using schema.sql
4. Run `Main.java`
